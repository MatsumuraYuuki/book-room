# AozoraLibrary

## サービス概要

青空文庫の作品を読みながら、読書記録や感想を管理できる読書記録サービスです。

### ペルソナ

青空文庫の作品を日常的に読むユーザーを想定しています。
特に、無料で公開されている文学作品を読むことが好きで、読んだ作品の記録や感想を残したいと考えている読書好きのユーザーを対象としています。

### 制作背景

青空文庫は多くの人に長年利用されている有名な電子書籍サービスですが、実際の利用シーンとしては「気になった作品を検索して読む」といった辞書やデータベースのような使われ方が多いのではないかと感じました。

しかし、青空文庫には数多くの名作文学が公開されており、読書体験としても非常に価値のあるコンテンツが揃っています。

そこで、

- 青空文庫の作品を継続的に読む
- 読んだ作品を記録する
- 感想を残す

といった読書体験をより豊かにするサービスを作りたいと考え、このアプリケーションを制作しました。

### 差別化した点

一般的な読書記録サービスは、紙の本や電子書籍など幅広い書籍を対象としています。

一方で本サービスでは、**青空文庫の作品に特化した読書記録サービス**として設計しました。

対象を青空文庫に限定することで、以下の点を特徴としています。

- 青空文庫の作品を簡単に検索できる
- 無料で読める文学作品に気軽に触れられる
- 青空文庫を日常的に利用するユーザーにとって使いやすい読書記録環境を提供する
